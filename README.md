# MedRitmo

Organizador de estudos para Medicina, feito para uma rotina com aulas, estágio e revisões realistas.

## Recursos

- Até duas matérias novas por dia.
- Revisões rápidas em rodízio.
- Revisão geral de todos os conteúdos no dia 1º.
- Planejador contínuo para o 6º período e semestres futuros, sem data final.
- Rotina semanal configurável, expandida automaticamente em quantas semanas forem necessárias.
- Checklist individual de estudo e revisão.
- Painel de questões com meta de 70%.
- Pomodoro de 40 minutos com trilha ambiente.
- Backup e restauração dos dados em arquivo JSON.

## Como abrir

Abra `index.html` em qualquer navegador moderno. Não é necessário instalar dependências.

## Publicar com GitHub Pages

Nas configurações do repositório, acesse **Pages**, selecione **Deploy from a branch**, escolha a branch `main` e a pasta raiz. O site passará a ter um endereço público do GitHub Pages.

## Privacidade

Os dados de estudo ficam no armazenamento local do navegador. O código publicado no GitHub não contém seus dados pessoais. Use o botão **Backup** para salvar uma cópia e **Restaurar** para recuperá-la.
